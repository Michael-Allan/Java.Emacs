Java Mode Tamed

Java Mode Tamed is a derived Emacs major mode designed to give the user more control over the Java mode that comes bundled with Emacs, particularly in regard to syntax highlighting.

  • User instructions.
  • Screen shots: untamed, tamed, and customized (as taming allows).

It defines user-customizeable faces for these features of Java:

  • Annotation components
  • Delimiters (quotes, various forms of bracket and separator)
  • Javadoc components
  • Keywords in various categories
  • Named literals: `true`, `false`, `null`
  • Package-declaration components
  • Type identifiers of type declarations, as opposed to references
  • Type variable declarations
