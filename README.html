Java Mode Tamed

Java Mode Tamed is a derived Emacs major mode designed to give the user more control over the Java mode that comes bundled with Emacs, particularly in regard to syntax highlighting. For this, it defines user-customizeable faces to highlight these features of Java:

  • Annotation components
  • Delimiters (quotes, various forms of bracket and separator)
  • Javadoc components
  • Keywords, according to role
  • Named literals: `true`, `false`, `null`
  • Package-declaration components
  • Shebang components * 
  • Type identifiers at point of declaration (as opposed to reference)
  • Type variables at point of declaration
